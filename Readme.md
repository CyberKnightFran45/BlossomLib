# BlossomLib

Es la librería núcleo del proyecto. Proporciona clases destinadas para el manejo de archivos, herramientas de cifrado, tipos serializables, entre otras utilidades que no se incluyen en las implementaciones estandar de .NET

Aquí se documentan únicamente los miembros públicos de la DLL y su respectivo enlace.

## Clases

- [SizeT](docs/SizeT.md) – Constantes y helpers para tamaños de datos y padding.
- [MathUtils](docs/MathUtils.md) – Funciones matemáticas y conversiones.
- [FileManager](docs/FileManager.md) – Lectura y escritura de archivos con seguridad.
